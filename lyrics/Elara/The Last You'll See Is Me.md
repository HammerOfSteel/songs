# The Last You'll See Is Me

**Labels:** Elara
**Created:** 2025-06-17
**Edited:** 2025-06-17

---

Oh, I wish it could go on forever,
What I hold with you, my heart.
But I promise when it all must end,
The last you'll see is me.
The last you'll hear is me.

Do you see the ice retreating, breaking?
Soon the bay runs free.
Once we sat where the red sun faded—
The last to burn were we.
The last to be were we.

You are my life.
If we only have hours to spare,
Then stay, love, and breathe with me there.
It’s never too late—
Though one must go,
There’s something more
In the new that waits for thee,
And the last you’ll see is me.

Now run through your fields of flowers,
The dusk begins to rise.
And when the night snuffs out the candle,
The last you’ll see is me.
The last you’ll hear is me.

Time will gather us like rain in oceans,
It won’t forget your name.
Should you fear this grave of stars that glisten,
The last you’ll see is me.
The last you’ll hear is me.

You are my life.
If we only have hours to spare,
Then stay, love, and breathe with me there.
It’s never too late—
Though one must go,
There’s something more
In the new that waits for thee,
And the last you’ll see is me.

My little light, my wildflower soul,
If the wind must take its toll,
Know I will wait, beyond the sea—
And the last you’ll see is me.

You are my life.
If we only have hours to spare,
Then live with me—just one more prayer.
It’s never too late—
Though one must go,
There’s something more
In the new that waits for thee,
And the last you’ll see is me.
Yes, the last you’ll see is me.
The last you’ll hear… is me.
