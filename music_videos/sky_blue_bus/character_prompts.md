## USAGE GUIDE

### How to Use These Character Sheets:

1. **Generate all character reference sheets FIRST** before creating individual video clips
2. **Use as anchor images** when prompting Grok Imagine for each clip - reference back to character sheet numbers
3. **Maintain consistency** by always describing characters with same features from these sheets
4. **Cross-reference** - when creating a clip with Narrator + Tima, use both Reference Sheets 1A-1D and 2A-2D
5. **Layer references** - for dream scenes, combine primary character sheets with folkloric character sheets

### Character Appearance Frequency:
- **Every clip (1-53):** Sky-blue bus (shown or referenced)
- **Clips 1-3, 6-53:** Narrator age 19-20
- **Clips 4-22, 24, 29, 52-53:** Tima age 3
- **Clips 40-43:** Young narrator age 12
- **Clip 1:** Two elderly pub men
- **Clips 45-48:** Folkloric creatures (trolls, giants, vittra, deer, goat)
- **Clip 37:** Father figure (implied/partial)
- **Clip 49:** Ghostly bus passengers
- **Clips 23-35, 52-53:** Cottage interior
- **Clips 3, 7-22:** Welsh outdoor locations

### Style Consistency Reminders:
- **Watercolor texture** - soft edges, gentle color bleeding
- **Warm color palette** - even in grey rainy scenes
- **Rounded shapes** - avoid harsh angles
- **Expressive eyes** - blue eyes especially important for narrator and Tima
- **Ghibli proportions** - realistic but slightly stylized, especially children
- **Children's book quality** - approachable, gentle, emotionally warm
